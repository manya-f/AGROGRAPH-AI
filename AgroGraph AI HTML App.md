# AgroGraph AI — Complete HTML Application

This block stores the full AgroGraph AI single-page application as an embedded HTML reference. The companion Python block (`Write AgroGraph AI HTML`) writes the complete file to disk.

## Sections Included (8 total)
1. **Hero Section** — Branding, tagline, CTA
2. **Sidebar** — Navigation & quick stats
3. **Plant Health & Weed Presence Analysis** — Upload interface & AI result panel
4. **Smart Recommendations** — Region/crop/issue action cards
5. **Savings Calculator** — Input form + projected savings output
6. **Charts** — Pesticide trends, yield loss, cost savings, env. impact
7. **Regional Intelligence** — Region selector & connector table
8. **Bottom Navigation** — Mobile-friendly tab bar

## Technology Stack
- Pure HTML5 / CSS3 / Vanilla JS (zero external dependencies)
- Zerve dark-theme design system (`#1D1D20` background, `#fbfbff` primary text)
- Responsive grid layout with CSS custom properties
- Chart.js-style canvas drawings via inline SVG placeholders
- Self-contained — single `.html` file, no CDN required at runtime

## Usage
Run the **Write AgroGraph AI HTML** Python block to generate `agrograph_ai.html` in the working directory. Open directly in any browser or serve via any static host.
